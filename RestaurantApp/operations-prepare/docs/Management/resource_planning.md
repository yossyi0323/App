# リソース計画

## 1. 人的リソース

### 1.1 開発チーム構成

- 開発者（1名）
  - コードレビュー
  - 要件定義
  - インフラ設計（AWS）
  - セキュリティ設計
  - 品質管理
- AIアシスタント（Cursor）
  - フロントエンド開発
    - Vue.js、TypeScript、Tailwind CSS
    - UI/UX設計
    - フロントエンドテスト（Vitest）
  - バックエンド開発
    - Java、Spring Boot
    - データベース設計
    - API設計・実装
    - バックエンドテスト
  - ドキュメント生成
    - 技術ドキュメント
    - ユーザードキュメント
  - インフラ実装支援
    - AWS環境構築のアドバイス
    - CI/CDパイプライン構築の支援
    - セキュリティ設定のアドバイス
    - モニタリング設定のアドバイス

### 1.2 作業時間

- 週末（土日）
  - 1日あたり10時間程度
  - 週10時間程度（予定が入る可能性あり）
- 平日夜間
  - 週5時間程度（2時間取れない週もあり）
- 合計
  - 週15時間程度
  - 月間約60時間程度
