# 共通サイドメニュー設計書

## 1. 画面概要

共通サイドメニューは、アプリケーション全体で使用されるサイドメニューコンポーネントです。

## 3. 画面項目定義

### 3.1 メニュー項目

- [トップページ](top_page_design.md)
  - アイコン: ホーム
  - リンク: トップページ
- [在庫確認業務](inventory_confirmation_design.md)
  - アイコン: 在庫
  - リンク: 在庫確認画面
- [補充（移動）業務](restocking_move_design.md)
  - アイコン: 移動
  - リンク: 補充（移動）画面
- [補充（作成）業務](restocking_create_design.md)
  - アイコン: 作成
  - リンク: 補充（作成）画面
- [発注依頼業務](order_request_design.md)
  - アイコン: 発注
  - リンク: 発注依頼画面
- [営業準備状況一覧](sales_preparation_design.md)
  - アイコン: 一覧
  - リンク: 営業準備状況一覧画面

### 3.2 メニュー項目のスタイル

- 高さ: 48px
- パディング: 左右16px
- アイコンサイズ: 24x24px
- フォントサイズ: 1rem
- アイコンとテキストの間隔: 16px

## 4. 表示/非表示制御

- ハンバーガーメニュークリックで表示/非表示
- 画面外クリックで非表示
- スライドアニメーション

## 5. レスポンシブ対応

- スマートフォンでの表示に最適化
- タッチ操作に適したサイズ
- スクロール可能な設計

## 6. ダークモード対応

- 背景色
  - ライトモード: 白
  - ダークモード: 暗いグレー
- テキスト色
  - ライトモード: 黒
  - ダークモード: 白
- アイコン色
  - ライトモード: 黒
  - ダークモード: 白
- ホバー色
  - ライトモード: 薄いグレー
  - ダークモード: より暗いグレー

## 7. アニメーション

- 表示/非表示時のスライドアニメーション
- ホバー時の色変更アニメーション
- クリック時のフィードバックアニメーション

## 8. アクセシビリティ

- キーボード操作対応
- スクリーンリーダー対応
- 適切なARIA属性の設定
- フォーカス時の視覚的フィードバック
