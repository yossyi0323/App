# GCal Keyboard Edit

Google Calendarの予定をキーボード操作だけで高速に編集・整理するためのGoogle Apps Script (Web App) ツールです。
マウスを使わず、キーボードだけで操作が可能です。

## 操作ガイド

### 1. ナビゲーション (移動)

| キー | 動作 |
| :--- | :--- |
| `←` / `→` | 日付（カラム）の移動 |
| `[` / `]` | **表示基準日の変更** (前日へ / 翌日へ) |
| `Tab` | 次の要素へ移動 (タイトル → 開始時刻 → 終了時刻 → 次の予定...) |
| `Shift + Tab` | 前の要素へ移動 |
| `↑` / `↓` | **[タイトル選択時]** 前後の予定へ移動<br>**[時刻選択時]** 時間を15分単位で増減 |

### 2. 時刻の調整 (Adjust)

選択中の要素に応じて、時間を直感的にずらすことができます。

| キー | 動作 | 詳細 |
| :--- | :--- | :--- |
| `j` | 時間を進める (+15分) | **タイトル選択時:** 開始・終了時間を*両方*ずらす（予定全体の移動）<br>**時刻選択時:** その時刻のみずらす |
| `k` | 時間を戻す (-15分) | 同上 |

### 3. アクション (操作)

| キー | 動作 | 詳細 |
| :--- | :--- | :--- |
| `c` / `n` | **Create** (新規作成) | 現在の列に新しい予定を作成します（デフォルトの開始時刻：現在時刻の15分丸め） |
| `d` | **Delete** (削除) | 選択中の予定を削除します |
| `Ctrl + z` | **Undo** (取り消し) | 直前の削除を取り消します（1件のみ） |
| `i` / `Enter`| **Input** (編集) | 選択中の項目（タイトルまたは時刻）を編集モードにします |
| `r` | **Refresh** (更新) | 最新のカレンダー情報を再取得します |
| `g` | **Go to Date** (ジャンプ) | 日付入力欄にフォーカスします (YYYY/MM/DD) |
| `t` | **Time** (現在時刻) | **[時刻選択時のみ]** 現在時刻（15分丸め）を入力します |
| `Alt + t` | **Today** (今日) | 表示基準日を今日にリセットします |

### 4. 日付操作 (Navigation by Date)

中央のカラムを基準日として、表示する3日間の範囲を自在に変更できます。

- `[` / `]` で1日ずつ移動。
- `g` を押すと画面中央の日付入力欄にジャンプし、任意の年月日を入力して `Enter` で確定することで、その日のカレンダーを表示できます。
- `Alt + t` でいつでも今日に戻れます。

### 5. 編集モード

`i` または `Enter` で編集モードに入ります。

#### タイトル編集
- テキストを入力して `Enter` または `Tab` で確定。
- `Tab` で確定すると、そのまま次のフィールド（開始時刻）へ移動します。

#### 時刻直接入力
以下のフォーマットで数字入力が可能です。

| 入力例 | 解釈される日時 |
| :--- | :--- |
| `1430` | 同日の 14:30 |
| `14:30` | 同日の 14:30 |
| `1/25 10:00` | 今年の 1月25日 10:00 |
| `2026/01/25 10:00` | 完全指定 |

> **Note**
> 日付を指定せず時刻のみ入力した場合、元の予定の日付が維持されます。
