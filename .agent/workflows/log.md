description: 作業ログの自動生成（AppとObsidian VaultのGit, AIとのChat, ObsidianのPeriodicNotesをもとに回答）
---

1. **対象期間の特定**:
   - ユーザーに「いつの作業ログを生成しますか？」と尋ねる。
   - 「昨日」「今日」などの相対日時や、「YYYY-MM-DD」などの指定を受け付ける。
   - 指定がない場合は、直近24時間をデフォルトとするか確認する。

2. **diff確認の原則**:
   - コミットが0件のときは未コミットの変更を確認する。コミットが少ないとき（おおむね1日分程度）は各コミットの内容も確認する。長期（1週間・1ヶ月など）のときは一覧程度に留め、各コミットの diff は見ない（膨大になるため）。
   - レポートの書き方はよしなに判断する。ファイル名の列挙だけで足りる場合もあれば、中身を見て要約したほうがよい場合もある。
   - 実行するコマンドはこのワークフローに書いていないものでもよい。`git log` / `git show` / `git diff` のオプションや、必要なら他のコマンドも自分で考えて実行する。
   - 1回コマンドを実行した結果、有用な情報が得られなかった場合や、より詳細を確認したほうがよい場合は、追加でコマンドを実行して詳細を確認する。例：コミットメッセージが「update」だけでは何をしたか分からないので、そのコミットの `git show` で差分を確認する。

3. **AppリポジトリのGitログ取得と詳細確認**:
   - `App` ディレクトリで、対象期間のコミット履歴を取得する。例：`git log --since="<START_TIME>" --until="<END_TIME>" --date=format:'%Y-%m-%d %H:%M:%S' --pretty=format:'%cd [%h] %s' --author="YOSHITO" --all --stat`
   - 上記「diff確認の原則」を踏まえ、必要に応じて `git status` / `git diff` / `git show <commit_hash>` など、適切なコマンドを自分で選んで実行する。

4. **Obsidian VaultリポジトリのGitログ取得と詳細確認**:
   - `Obsidian Vault` ディレクトリで、対象期間のコミット履歴を取得する。原則は App と同様。実行するコマンドはよしなに判断する。

5. **Periodic Notesの確認**:
   - 指定期間に対応するDaily Note（例: `Periodic Notes/01.Daily/YYYYMMDD.md`）の内容を確認する。
   - その日に追記されたタスクやメモがあれば抽出する。
     - ※1ヶ月など長期にわたり確認する場合にファイルが多すぎる（1ヶ月ならMonthlyとWeekly4つとDailyが30日分など）の場合は、WeeklyやMonthlyなど適切な粒度での確認に留めること。
   - **Daily Note**: 指定期間に含まれるすべての日次ノート（`Periodic Notes/01.Daily/YYYYMMDD.md`）を確認する。
   - **Weekly Note**: 期間が「1週間（7日）」以上の場合は、該当する週次ノート（`Periodic Notes/02.Weekly/YYYY-Www.md`）も確認する。
   - **Monthly Note**: 期間が「1ヶ月（28日）」以上の場合は、該当する月次ノート（`Periodic Notes/03.Monthly/YYYY-MM.md`）も確認する。
   - QuarterlyやYearlyも同様。各ノートから、実施したタスク、振り返り、追記されたメモを抽出する。

6. **チャット履歴の統合**:
   - あなた自身の記憶（直近の会話ログ）を参照し、その期間にどのような議論や試行錯誤があったかを振り返る。
   - Gitログのコミットメッセージだけでは分からない「意図」や「苦労した点」を補足する。

7. **レポート生成**:
   - 収集した情報を時系列に整理し、以下のフォーマットで出力する。Git の変更は、ファイル名だけで足りる場合も、要約や補足が必要な場合も、よしなに判断して記載する。

   ## [YYYY-MM-DD] 作業ログ

   ### 概要
   (1-2行でのサマリ)

   ### タイムライン
   | 時刻 | ソース | 作業内容 | 詳細/コミット |
   | :--- | :--- | :--- | :--- |
   | HH:MM | App (Git) | ... | ... |
   | HH:MM | Chat | ... | ... |
   | HH:MM | Obsidian | ... | ... |

   ### 詳細・考察
   (AIによる補足説明)
