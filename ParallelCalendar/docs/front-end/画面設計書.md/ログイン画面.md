# ログイン画面

## 画面概要

### ログイン

シンプルなログイン画面。

Google認証を通すとカレンダー画面に遷移する。

各画面のヘッダーのログアウトボタンからログアウトし、ログイン画面に遷移する。

```tsx
<Login />
```

### URL

/login のみアクセス可能。


## 画面要素

1. 共通ヘッダー
2. アプリアイコン
3. アプリ名
   1. 「Parallel Calendar」
4. Google認証ボタン
   1. 「Googleでログイン」

## イベント

1. Google認証ボタンをクリックすると、Google認証画面に遷移する。
2. Google認証画面でログインすると、カレンダー画面に遷移する。
3. カレンダー画面でログアウトすると、ログイン画面に遷移する。
