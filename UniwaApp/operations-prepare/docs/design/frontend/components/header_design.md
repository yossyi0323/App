# 共通ヘッダー設計書

## 1. 画面概要

共通ヘッダーは、アプリケーション全体で使用されるヘッダーコンポーネントです。

## 2. 画面項目定義

### 2.1 ハンバーガーメニュー

- アイコン: ハンバーガーメニュー
- クリックで[サイドメニュー](side_menu_design.md)を表示
- ダークモード対応

### 2.2 ページタイトル

- 表示: 現在のページ名
- フォントサイズ: 中
- ダークモード対応

## 3. レスポンシブ対応

- スマートフォンでの表示に最適化
- タッチ操作に適したサイズ

## 4. ダークモード対応

- 背景色
- テキストの色
- アイコンの色

## 5. アニメーション

- ハンバーガーメニュークリック時のスムーズな遷移
- サイドメニュー表示/非表示時のスライドアニメーション

## 6. アクセシビリティ

- キーボード操作対応
- スクリーンリーダー対応
- 適切なARIA属性の設定
