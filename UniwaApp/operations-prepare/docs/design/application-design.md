# アプリケーション設計書

## 1. 画面設計

### 1.1 トップページ
- 画面概要
  - 業務日付の入力と予約状況のメモを管理する画面
  - 各業務画面へのナビゲーションを提供
- レイアウト
  - ヘッダー
    - ハンバーガーメニュー（サイドパネル表示用）
    - アプリケーションタイトル「翌日営業準備管理」
  - サイドパネル
    - 各業務画面へのリンク
    - テーマ切り替え（ライト/ダークモード）
  - メインコンテンツ
    - 業務日付選択（v-date-picker）
    - 予約状況メモ入力エリア
    - 各業務画面へのナビゲーションボタン
- 実装技術
  - Vue.js + Vuetify
  - レスポンシブデザイン対応
  - マテリアルデザイン準拠

### 1.2 サイドパネル
- 機能
  - 各業務画面へのナビゲーション
  - テーマ切り替え（ライト/ダークモード）
- 実装技術
  - Vuetifyのv-navigation-drawerコンポーネント
  - テーマ切り替えはVuetifyのuseThemeを使用

## 2. テーマ設計
- カラーテーマ
  - プライマリカラー: #e07833
  - ライトモード/ダークモード対応
- フォント
  - 日本語フォント: "Helvetica Neue", Arial, "Hiragino Kaku Gothic ProN", "Hiragino Sans", Meiryo, sans-serif
- タイポグラフィ
  - 見出し（h1-h6）と本文のスタイル定義
  - レスポンシブ対応

## 3. メッセージ管理
- メッセージカテゴリ
  - ERROR: エラーメッセージ
  - SUCCESS: 成功メッセージ
  - WARNING: 警告メッセージ
  - NOTIFY: 通知メッセージ
- 実装方式
  - 共通部分を再利用
  - 変数部分を${}で動的に埋め込み
  - TypeScriptの型安全性確保
