# 一括操作バー（bulk-action-bar.tsx）テスト仕様書

## 1. テスト概要
一括操作バーコンポーネントの主要機能をテストし、以下の点を確認します：
- ボタンの表示
- コールバック関数の実行
- 場所名の表示
- ボタンの無効化状態

## 2. テストケース

### 2.1 初期表示テスト
#### 確認条件
- コンポーネントを表示した時の初期状態

#### 想定結果
- 「全て確認済に」ボタンが表示される
- 「全て要補充に」ボタンが表示される
- 場所名が指定されている場合、場所名が表示される
- 場所名が未指定の場合、場所名が表示されない

### 2.2 ボタン操作テスト
#### 確認条件
- 「全て確認済に」ボタンをクリックした時

#### 想定結果
- `onMarkAllChecked`コールバックが1回呼ばれる

#### 確認条件
- 「全て要補充に」ボタンをクリックした時

#### 想定結果
- `onMarkAllNeedsRestock`コールバックが1回呼ばれる

### 2.3 場所名表示テスト
#### 確認条件
- `locationName`プロパティに値が設定されている場合

#### 想定結果
- 場所名が正しく表示される
- 場所名のスタイルが適用される

#### 確認条件
- `locationName`プロパティが未設定の場合

#### 想定結果
- 場所名が表示されない
- ボタンのみが表示される

## 3. テストデータ
### 3.1 モックデータ
```typescript
const mockProps = {
  onMarkAllChecked: jest.fn(),
  onMarkAllNeedsRestock: jest.fn(),
  locationName: '倉庫A'
};
```

## 4. テスト実行手順
1. テスト環境のセットアップ
   ```bash
   npm install
   npm test
   ```

2. 特定のテストの実行
   ```bash
   npm test bulk-action-bar.test.tsx
   ```

## 5. 注意事項
- コールバック関数のモックが正しく設定されていることを確認
- ボタンのクリックイベントが正しく発火することを確認
- スタイリングが正しく適用されていることを確認
